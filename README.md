#Student Records Manager (CRUD)
This is a Django-based web application designed to manage student records in a database. It allows users to perform core database operations through a clean web interface.

###Current Features
*Create: Add new student profiles including names and email addresses.

*Read: View a complete list of all students currently stored in the database.

*Update: Edit existing student information if details change.

*Delete: Remove student records permanently from the system.

###Technical Implementation
*Django ORM: Used for interacting with the SQLite database without writing manual SQL.

*ModelForms: Used for secure and automated HTML form generation and validation.

*Function-Based Views (FBVs): Handles the logic for each database operation.

*CSRF Protection: Standard Django security implemented for all form submissions.
